# Event photos (gallery)

The "See you in the room" gallery near the bottom of the page shows a photo mosaic. Drop event photos here named:

- `1.jpg` — the large feature tile (shows biggest, so pick your best wide shot)
- `2.jpg`
- `3.jpg`
- `4.jpg`
- `5.jpg`

Notes:
- `1.jpg` fills a large 2x2 tile; the rest are smaller squares. Landscape works for all of them.
- Any missing number just shows a subtle gradient tile instead of breaking.
- Candid shots of people at deal rooms, dinners, and happy hours work best. They're cropped to fill, so don't worry about exact dimensions.
