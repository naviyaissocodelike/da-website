# Market report PDFs

The News & Reports page (`news.html`) links each report card to a PDF here. Export each report from the old site (or Google Doc) as a PDF and drop it in with these exact names:

- `drones-uas.pdf` — Drones & Unmanned Aerial Systems (Yousof Omeish)
- `ai-workforce.pdf` — AI-Powered Upskilling & Workforce Transformation (Catherine McMillan)
- `aquaculture.pdf` — Emerging Technologies in Aquaculture (Tony Lodge)
- `humanoid-robots.pdf` — General Purpose Humanoid Robots (Juliano Perczek)
- `biomaterials.pdf` — From Biomaterial Implants to Regenerated Tissues (Allison Horenberg)
- `digital-therapeutics.pdf` — Digital Therapeutics (Courtnie Phillip)

Until a PDF is present, that report's "Read report" link will 404. The card itself (title, author, blurb) shows regardless.

Why PDFs: it hosts the reports on your own domain (so they survive the domain switch off the old Framer site), keeps the original formatting exactly, and avoids re-typing the long report bodies. If you'd rather host them as full web pages instead, send me the text for any report and I'll build it as its own page.
