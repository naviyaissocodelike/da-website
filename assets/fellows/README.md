# Fellow headshots

Headshots for Halo Fellowship cohort members go here. Use a slugged version of each fellow's name, for example:

- `yousof-omeish.jpg`
- `catherine-mcmillan.jpg`
- `tony-lodge.jpg`
- `juliano-perczek.jpg`

Square photos look best (shown in small circles, cropped to fill), at least 200×200px, face centered. Until a photo is present, the cohort slot shows the fellow's initials.

Once you share the full cohort lists, I'll wire each fellow's circle to its photo (and a LinkedIn link if you want).
