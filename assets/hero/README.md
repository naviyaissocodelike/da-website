# Hero slideshow images

The hero background cycles through up to 6 images. Drop your photos here named exactly:

- `1.jpg`
- `2.jpg`
- `3.jpg`
- `4.jpg`
- `5.jpg`
- `6.jpg`

How it behaves:
- Images crossfade slowly, one every ~5.5 seconds, behind a light wash so the dark headline stays readable.
- You don't need all six. Any that are missing are skipped automatically, so 2, 3, or 4 images work fine too.
- If none are present, the hero just shows the light gradient.
- For visitors who set "reduce motion," it shows the first image without cycling.

What looks best: wide, landscape photos (2000px+ across), ideally your event shots. Bright or busy is fine, the wash handles it. If a photo is a `.png`, either convert it to `.jpg` or change that image's extension in `index.html`.
