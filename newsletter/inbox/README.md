# Inbox

Un-sent newsletter items live here. One file per item. Copy a `_TEMPLATE-*.md`, fill it in (rough notes are fine), commit. `/newsletter` sweeps this folder when compiling the next issue and archives processed items into `../issues/`.

Files starting with `_` are templates and are never included in an issue.

⚠️ This repo is public — only log deal details the founder has approved for community distribution.
