# Issues

One file per newsletter issue, named `YYYY-MM.md` (e.g. `2026-08.md`). Each file is the reviewed draft — subject line options, preview text, and full section copy. The rendered Mailchimp HTML, when generated, sits alongside as `YYYY-MM.html`.

Items that went into a sent issue are archived under `YYYY-MM-items/` so the inbox stays clean and nothing gets double-sent.
